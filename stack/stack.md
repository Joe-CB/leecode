

# Stack 栈

1. 特性： LIFO(Last in first out)
2. 适用于需要记录之前的状态
3. 不像array，不能随机访问

> DP 一般要求O(1)复杂度访问之前的状态，所以一般用array，Stack 的状态只能获取一次，获取后就扔掉了
